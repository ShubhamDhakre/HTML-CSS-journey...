Syntax: @font-face { font-family: 'MyFont'; src: url('myfont.woff2') format('woff2'); font-display: swap; /* or auto, block, fallback, optional */ } Values of font-display: auto – Default behavior. The browser decides how to handle the font loading. block – Hides text until the font is fully loaded (can cause a "flash of invisible text"). swap – Shows fallback font immediately, then swaps to the custom font when it's ready (causes "flash of unstyled text"). fallback – Shows fallback font for a short time, then swaps to custom font only if it’s ready quickly. optional – Similar to fallback, but the font may never load on slow connections or if performance is prioritized. Example: @font-face { font-family: 'Roboto'; src: url('roboto.woff2') format('woff2'); font-display: swap; } Use font-display: swap; if you want better performance and user experience.
